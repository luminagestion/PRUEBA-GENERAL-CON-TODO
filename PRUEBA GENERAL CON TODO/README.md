# LUMINA-MAPA

Prototipo de directorio y mapa de artistas 🎶  
Proyecto desarrollado con **React + Vite + TailwindCSS**.

---

## 🚀 Instalación y uso

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/luminagestion/LUMINA-MAPA.git
   ```

2. **Entrar en la carpeta del proyecto**
   ```bash
   cd LUMINA-MAPA
   ```

3. **Instalar dependencias**
   ```bash
   npm install
   ```

4. **Levantar el servidor de desarrollo**
   ```bash
   npm run dev
   ```

5. Abrir en el navegador la URL que muestra Vite (por defecto `http://localhost:5173`).

---

## 🛠️ Stack Tecnológico
- [React](https://reactjs.org/) – Librería principal de UI  
- [Vite](https://vitejs.dev/) – Bundler / Dev Server  
- [TailwindCSS](https://tailwindcss.com/) – Estilos con utilidades  
- [shadcn/ui](https://ui.shadcn.com/) – Componentes UI  
- [lucide-react](https://lucide.dev/) – Íconos  

---

## 📂 Estructura principal
- `src/` → componentes y vistas  
- `public/` → archivos estáticos  
- `package.json` → dependencias y scripts  
- `vite.config.js` → configuración de Vite  

---

## 🔑 Notas
- **Node recomendado:** v18 o v20  
- Si hay un archivo `.env`, crear un `.env.local` con tus propias variables.  
- Para generar la versión lista para producción:
  ```bash
  npm run build
  ```
